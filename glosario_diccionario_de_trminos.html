Glosario. Diccionario de términos

Glosario

A continuación, se definen una serie de términos que el alumnado debe conocer y que observará a lo largo de la situación de aprendizaje:

  • Dato estadístico: Valor numérico que describe una característica de una muestra o población. 
  • Media: Valor representativo de un conjunto de datos numéricos que se obtiene sumando todos los valores y dividiendo el resultado entre el número total de datos.
  • Desviación típica: Medida que indica cuánto se alejan los datos de la media aritmética. 
  • Varianza: Parámetro estadístico que mide la dispersión de los datos respecto a la media. Se calcula como el promedio de los cuadrados de las desviaciones. Una varianza alta indica que las opiniones o datos del alumnado son muy heterogéneos.
  • Frecuencia absoluta: Número de veces que se repite un valor en un estudio. 
  • Muestra: Subconjunto representativo de una población que se selecciona para realizar un estudio. 
  • Muestreo sesgado: Error que ocurre cuando la muestra elegida no representa fielmente a toda la población. 
  • Error muestral: Es la diferencia que existe entre los resultados obtenidos de una muestra y los resultados que se obtendrían si se encuestara a toda la población. 
  • Parámetros de dispersión: Valores numéricos (como el rango, la varianza o la desviación típica) que informan sobre la variabilidad de los datos.
  • Población: Conjunto total de individuos o elementos que se desean estudiar. 
  • Porcentaje: Forma de expresar un número como una fracción de 100.
  • Variable cualitativa: Tipo de variable que expresa una cualidad o categoría no numérica. 
  • Variable cuantitativa: Tipo de variable que se expresa mediante cantidades numéricas y con la que se pueden realizar operaciones aritméticas.
